📊 Sistema de Cálculo de Cashback

Aplicação web full stack desenvolvida para calcular cashback de compras, considerando regras de desconto, tipo de cliente (VIP ou comum) e histórico de consultas.

O projeto é dividido em:

🔙 Back-end: API em Python com Flask
🌐 Front-end: Interface em HTML, CSS e JavaScript


🚀 Funcionalidades
Cálculo automático de cashback
Aplicação de cupom de desconto
Regra especial para clientes VIP
Dobro de cashback para compras acima de R$500
Registro de histórico por IP (últimas 4 consultas)

🛠️ Tecnologias que utilizei

Back-end:
Python
Flask
Flask-SQLAlchemy
Flask-CORS

Front-end:
HTML5
CSS3
JavaScript
